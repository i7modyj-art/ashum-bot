# 🤖 Ashum Trading Bot

بوت تيليقرام ذكي يفحص السوق الأمريكي تلقائياً 8 مرات يومياً ويرسل الفرص بتحليل Claude AI.

## ✨ الميزات

- 🔍 **فحص تلقائي** 8 مرات/يوم بتوقيت السوق الأمريكي
- 🤖 **تحليل Claude AI** - ICT/SMC + Price Action متعدد الإطارات
- 📊 **بيانات متعددة الأطر**: 15M, 1H, 4H, Daily
- ⚡ **استراتيجيتان**: Swing Trading + 0DTE Options
- 🎯 **شروط صارمة**: Swing ≥ 75% ثقة، 0DTE ≥ 85% ثقة
- 💰 **مجاني تماماً** على GitHub Actions (عدا Claude API)
- 🎮 **أوامر تفاعلية** عبر تيليقرام

## 📋 الأوامر

- `/help` أو `مساعدة` - دليل الأوامر
- `/analyze AAPL` أو `تحليل AAPL` - تحليل عميق بـ AI
- `/scan` أو `فحص` - مسح فوري لكل القائمة
- `/list` أو `قائمة` - عرض الأسهم المتابعة
- `/quote AAPL` أو `سعر AAPL` - السعر الحالي
